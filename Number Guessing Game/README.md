# Python Number guessing game.

by: Roadmap.sh

https://roadmap.sh/projects/number-guessing-game

i did this by importing randint in random lib and creating a class with only one method, play, and then i did a while-loop to start the game, 2 base numbers the secret number and the tries counter.
After this, i started another while-loop catching the **_ValueError_** and asking the number there is between 0 and 100, if your number are bigger or lower than the secret number the code will tell you and increase the counter.
And if you hit the right number the code will print that you hit it, the secret number, and the number of tries.

I did a continue system that look at the user input and if it is yes, y, Yes, Y , YES. The game will stop
